### 4. Script Loading Order
- bootstrap.bundle.min.js

- i18n.js
- script.js

## How It Works:

1. When page loads, `i18n.js` initializes with saved language (default: 'en')
2. `currentLanguage` global variable is accessible to all scripts
3. `hospital-names.js` provides `getHospitalNameInLanguage()` function
4. When user selects language from dropdown, `switchLanguage()` is called
5. `switchLanguage()` calls `renderGrid()` to refresh all cards with new language names
6. Detail modal also uses `currentLanguage` when opened
